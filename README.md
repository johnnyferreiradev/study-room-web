# Study Room

## Requisitos

  - Node
  - Gerenciador de pacotes javascript: npm ou yarn

 ## Instalação de dependências

 ###### Entrar na pasta raiz do projeto e digitar o seguinte comando:

 Usando npm:
 ```
 npm install
 ```

  Usando yarn:
 ```
 yarn
 ```

 ## Iniciar a aplicação

  Usando npm:
 ```
 npm run start
 ```

  Usando yarn:
 ```
 yarn start
 ```
 ## Porta
  O servidor é executado por padrão na porta 3000 (http://localhost:3000)
